# 🚀 친구들과 함께 쓰는 실시간 노션 위젯 기획서 & 기능 소개서

> **"카톡방에서 흘러가는 일정과 할 일은 그만!"**
> 친구들과 함께 공유하는 노션 페이지를 더욱 풍성하고 가치 있게 만들기 위해 제작된 **Supabase/Firebase 기반 실시간 인터랙티브 노션 위젯** 기획 및 기능 소개서입니다.

---

## 1. 프로젝트 개요 (Overview)
* **프로젝트명:** 소모임/친구 공유용 실시간 노션 위젯 (Notion Real-Time Widgets)
* **기획 배경:** * 카카오톡 단톡방에서의 대화는 휘발성이 강해 중요한 일정 공유나 할 일 관리가 어려움.
  * 노션 공유 페이지를 개설했으나, 텍스트 위주의 정적 기능만으로는 실시간 소통과 재미 요소를 충족하기 부족함.
  * 실시간 통신 기술을 결합하여, 노션 안에서 친구들의 상태와 반응을 즉각적으로 확인하고 인터랙션을 나눌 수 있는 전용 위젯을 개발함.
* **주요 타겟:** 노션으로 동아리, 팀 프로젝트, 혹은 친구들과 일상/일정을 공유하는 소규모 그룹 (4인 기준 최적화)

---

## 2. 공통 시스템 아키텍처 및 핵심 기능

### 📡 실시간 연결성 및 동기화 (Real-Time Sync)
* **상태 표시기 (Status Indicator):** * 모든 위젯 상단에 실시간 데이터베이스 연결 상태가 표시됩니다.
  * 로딩 및 연결 중일 때는 **`🔥 연결 중`**으로 표시되며, 정상적으로 세션이 수립되면 **`✅ 실시간 연결됨`**으로 상태가 실시간 전환됩니다.
* **4인 동시 동기화:** * 한 명이 위젯에 데이터를 입력하거나 반응을 보이면, 웹소켓 및 실시간 Pub/Sub 채널을 통해 나머지 3명의 화면에도 **별도의 새로고침 없이 즉시(Instant)** 반영됩니다.

---

## 3. 위젯별 상세 기능 소개 (Widget Specifications)

### 📝 1. 실시간 타임라인 위젯 (TimeLine Widget)
친구들의 소소한 소식, 한 줄 방명록, 혹은 중요한 공지사항을 실시간으로 피드 형태로 공유하는 공간입니다.
* **데이터 영속성:** 게시글이 백엔드 데이터베이스(Firebase/Supabase)에 안전하게 저장되어, 노션 페이지를 닫거나 새로고침해도 과거 기록이 그대로 유지됩니다.
* **피드 최적화:** 화면 공간을 효율적으로 활용하고 성능을 유지하기 위해 **최신 50개**의 게시글까지만 제한하여 동적으로 표시합니다.
* **활용 예시:** "나 지금 카페 도착!", "오늘 7시 홍대 모임 늦지 마라~" 등의 실시간 한 줄 소통.

### 📊 2. 오늘의 기분 트래커 위젯 (Mood Tracker Widget)
서로의 감정 상태를 말로 묻지 않아도 한눈에 파악하고 챙겨줄 수 있는 감정 공유 위젯입니다.
* **실시간 감정 점수 공유:** 내가 선택한 오늘의 기분 점수나 이모지가 즉각 친구들의 위젯에 업데이트됩니다.
* **히스토리 관리:** 감정 기록의 흐름을 볼 수 있도록 **최신 30개**의 기록을 리스트 형식으로 유지합니다.
* **통계 자동 반영:** 친구들의 기분 점수를 기반으로 한 평균 점수나 가장 많은 기분 상태 등의 **통계 데이터가 백엔드에서 계산되어 화면에 자동 반영**됩니다.

### 🗳️ 3. 밸런스 투표 위젯 (Balance Vote Widget)
저녁 메뉴 고르기, 모임 장소 정하기 등 소소한 결정 장애를 실시간 투표로 재치 있게 해결하는 위젯입니다.
* **실시간 결과 집계:** 투표 버튼을 누르는 순간 표 수와 그래프가 실시간으로 모든 멤버에게 동기화됩니다.
* **토글형 투표 취소:** 내가 누른 버튼을 **다시 한 번 누르면 투표가 취소**되는 직관적인 토글 시스템을 지원합니다.
* **스마트 선택지 변경:** 이미 A안에 투표한 상태에서 B안(반대편 선택지)을 누르면, **이전 투표(A안)가 자동으로 취소되고 새 선택지(B안)로 표가 이동**하여 중복 투표를 방지합니다.

## 4. 확장 기능 및 연동 계획
현재 구축된 실시간 통신 규격 및 레이아웃을 바탕으로, 카톡방의 단점을 완벽히 보완할 수 있는 추가 위젯 연동을 목표로 합니다.
* **공유 캘린더 위젯:** 각자의 일정을 등록하고 실시간으로 병합하여 볼 수 있는 타임테이블 뷰.
* **공동 할 일(To-Do) 리스트 위젯:** 친구들과 분담한 역할이나 장보기 목록을 체크하면 실시간으로 취소선이 그어지는 협업형 체크리스트.
